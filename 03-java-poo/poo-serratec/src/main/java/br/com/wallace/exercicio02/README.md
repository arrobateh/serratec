### 💻 Exercício
#### Classes e Objetos
##### Tarefa: Criando a classe ContaBancaria
Detalhes: Migrar a lógica do exercício anterior para uma estrutura Orientada a Objetos. Criar a classe com os atributos numero, titular e saldo, e os métodos depositar() e sacar(). Instanciar no main.

Regra de Negócio:
- Todas as regras do exercício anterior devem ser seguidas também;
- Toda conta nova aberta ganha um bônus de boas-vindas de R$ 50,00 (isso deve ser garantido no momento da criação do objeto/construtor).

### 🚀 Aprendizados
- Aprendi a criar classes e trabalhar com informações encapsuladas, alem de aplicar os conceitos de objetos e métodos para organizar melhor o código e facilitar a manutenção.
